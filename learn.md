# 📝 Todo Fetcher - Simple API Integration

This is my first project where I successfully integrated a REST API to fetch and display data. The project is built using vanilla JavaScript and focuses on the core logic of data fetching and DOM manipulation.

## 🎯 Project Goal
The main objective was to create a functional "Load Todo" button that retrieves all 200 items from a dummy API and renders them directly on the webpage.

### ✨ Key Features
* **API Connection:** Fetches real-time data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/todos).
* **One-Click Execution:** All 200 todos load instantly upon clicking the button.
* **Dynamic Rendering:** Uses JavaScript to create and append elements for each todo item.
* **Clean Logic:** Focuses purely on functionality without any external CSS or styling.

## 🛠️ Technologies Used
* **HTML5:** For the basic page structure and the load button.
* **JavaScript (ES6):** For the `fetch()` API, `.then()` promises, and DOM methods.

## 🚀 How to Use
1. Open the `index.html` file in your browser.
2. Click the **"Load Todos"** button at the top of the page.
3. Scroll down to see the list of 200 todo items including their titles and completion status.

## 🧠 Core Learning Points
* Implementing the `fetch()` method to get data from a URL.
* Converting API responses into JSON format.
* Using `forEach` loops to iterate through a large dataset (200 items).
* Using `innerHTML` to dynamically update the webpage content.
